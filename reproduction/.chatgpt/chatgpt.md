Clark 这篇本身也是这种逻辑：它不是先发现一个大数据集再做一堆统计，而是先把问题定义为“异质 tuning curve 是否仍可由 continuous-attractor mechanism 产生”，然后用数据构造 RNN、用 Gaussian-process generative process 扩大网络规模、最后做 large-(N) 和 DMFT 分析。文章自己也把流程画成 data-derived RNN → generative process → synthetic data-derived RNN → DMFT，而不是“纯数据分析项目”。 这对你是重要启发：你可以不拥有一手实验数据，也可以做一篇以理论模型为中心的工作。

你现在应当选择一个“最低可行课题”，而不是继续扩散到 head direction、grid、place、odor、aggression、history bias 全部方向。我建议主线定为：

“Can local synaptic plasticity learn heterogeneous continuous-attractor manifolds generated by Gaussian-process tuning statistics?”

中文就是：是否存在一种局部学习规则，使随机初始化的 RNN 在结构化低维输入驱动下，学习出由 GP 生成的异质连续吸引子流形。

这个题目有几个优点。第一，它直接承接 Clark：Clark 是离线优化 (J)，你做的是把 (J_{\mathrm{opt}}) 的构造改造成 local learning。第二，它不强依赖公开数据集，因为 GP tuning statistics 本身就是模型生成器。第三，它有清楚的可检验指标：流形误差、Jacobian 谱、normal/tangential timescale separation、bump 是否稳定、是否出现二重简并或近似低秩结构。第四，它可以天然写进 9 月开题：背景、缺口、方法、初步结果、后续扩展都成立。

具体来说，你的主线不要从真实数据开始，而从 synthetic teacher manifold 开始：

[
\theta\in[0,2\pi)
\rightarrow x_i^\star(\theta)
\rightarrow \phi_i^\star(\theta)
\rightarrow J_{\mathrm{target}}.
]

Clark 的离线做法是给定目标流形 (\phi_i^\star(\theta))，要求在流形上满足

[
\tau \dot x_i=-x_i+\sum_j J_{ij}\phi_j+b=0,
]

然后用最小二乘求 (J)。他们明确指出这个解含有类似 Hebbian 的结构，即 postsynaptic (x_i^\star(\theta)) 和 presynaptic (\phi_j^\star(\theta')) 的乘积，但中间包含 (K^{-1}_{i,\lambda})，所以它不是生物意义上的在线局部学习。 你的研究切口就是把这个“离线伪逆解”替换成“在线局部更新规则”。

第一阶段只做一个 toy model，规模控制在 (N=128) 或 (N=256)，角度离散点 (N_\theta=100) 或 (128)。不要一上来碰 DANDI，不要一上来做 NWB，不要一上来做大工程。你的 5070Ti + 9800X3D 完全足够。每次仿真都应该在几分钟内跑完，否则就说明模型设计过重。

你可以先用三种学习规则比较：

[
\Delta J_{ij}
=============

\eta e_i \phi_j-\eta\lambda J_{ij},
]

其中

[
e_i=x_i^{\mathrm{target}}-\sum_kJ_{ik}\phi_k.
]

这是最接近 supervised local delta rule 的版本。它不是 backprop，因为每个突触只用 postsynaptic error (e_i)、presynaptic activity (\phi_j) 和自身权重 (J_{ij})。

第二种是更 Hebbian 的版本：

[
\Delta J_{ij}
=============

\eta x_i\phi_j-\eta\alpha J_{ij}.
]

第三种是带 homeostasis 或 Oja-like normalization 的版本：

[
\Delta J_{ij}
=============

\eta x_i\phi_j-\eta \alpha x_i^2 J_{ij}.
]

你真正要看的不是哪一个“更像大脑”，而是哪一个可以让网络从 (J_0) 走到一个动力学上等价于 (J_{\mathrm{opt}}) 的区域。科研上这就够了：local rule 不一定逐元素复现 Clark 的 (J)，只要复现吸引子动力学即可。

关于 place cell，我建议暂时降级为副线，不要作为主线。Mainali 这篇确实适合你，因为它的模型就是“随机 Gaussian process + threshold/rectification 生成 place fields”，而且他们强调 CA1 的输入可以被建模为 Gaussian process，可能反映随机投射结构。 但如果你现在直接去找 place-cell 原始数据，你会陷入数据泥潭。更好的做法是：先在 synthetic 2D place field 上验证你的 local-learning RNN 是否能学到二维流形。等这个做通，再决定是否拿 Mainali 或其他公开数据做一小节验证。

因此，place cell 在你的路线中不是“我要找新数据集复现 place field 论文”，而是：

[
\mathbf r\in\Omega\subset\mathbb R^2
\rightarrow x_i^\star(\mathbf r)\sim \mathcal{GP}(0,\Gamma(\mathbf r,\mathbf r'))
\rightarrow \phi_i^\star(\mathbf r)
\rightarrow \text{learned RNN manifold}.
]

这与 Clark 的 higher-dimensional extension 是同一类问题。Clark 文章已经把 grid-cell 模型扩展到二维 toroidal attractor，并强调连续变量的流形可以通过异质 tuning 统计和大 (N) 分析恢复经典吸引子机制。 你要做的是把“构造 (J)”变成“学习 (J)”。
